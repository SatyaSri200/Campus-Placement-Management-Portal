Campus Placement Management Portal

Overview
The *Campus Placement Management Portal* is a Django-based web application to manage campus recruitment activities.  
It enables *students to register, upload resumes, and track applications, while **admins can manage student data and monitor selections*.

*Technologies Used:*
- Backend: Python, Django  
- Frontend: HTML, CSS, JavaScript  
- Database: SQLite (local development)  



 Features

 Student
- Registration & login  
- Profile management  
- Upload resume and profile picture  
- View application status  

Admin
- Admin login  
- Manage student registrations  
- Track selected students  
- View uploaded resumes
